Section II Compare Operating System Diffrences

Both OS X and Fedora 20 have bash so there where a lot of similarities but some key differences.

Similarities

  • Same command to fork the process (command > output-file.log) &
  • Both used java to run and javac to compile Java programs.
  • Both support log files natively.

Differences

  • Different flag on ps command. OS X uses ps -M to show process and their threads, while Fedora 20 uses ps -m. Note the lower case "m".
  • Threads are printed differently. While Fedora 20 used a "-" instead of repeating the same process ID for each thread of a process, OS X list the process ID for each thread repeatedly, kind of messy and less clean.
  • Mac does not have shortcuts to cycle through previous commands starting with the letters the user started typing. It depends on the reverse search tool in bash which is clunky.
