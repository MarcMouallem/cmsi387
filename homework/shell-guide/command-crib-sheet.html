Command Crib Sheet A Unix command glossary

Bash, standing for born again shell, is the Linux command line shell. Command line is the most powerful and efficient way to interface with applications.

Unlike most application interfaces, the command line does not require the existence of multiple and specific forms, buttons or any other interfacing widgets. Instead it provides a single text input that can specify any command for any application that currently, or will ever, exist. This is what makes command line so powerful.

In addition, there is no sense of navigating the cursor in a terminal. For that reason the terminal is always ready to accept the users next command. This is what makes the command line so efficient.

Section I Navigation

The most basic skill in Bash, is the ability to navigate directors as well as read a create files within them.

Action Basic Invocation Explanation
List files and directories ls List all the files and directories in the current directory.
Change directory cd my-directory Changes current directory to directory named my-directory.
Make directory mkdir my-directory Create a new directory named my-directory in the current directory.
Delete directory rmdir my-directory Deleted directory named my-directory in the current directory.
Read file cat my-file.txt Short for concatenate, prints content of file named my-file.txt to the console.
Create file touch my-file.txt Creates file named my-file.txt in current directory.
Delete file rm my-file.txt Deletes file named my-file.txt in current directory.
Copy cp my-file.txt nested-directory Copies file named my-file.txt into directory named nested-directory. Can be used with files or directories.
Move mv my-file.txt nested-directory Moves file named my-file.txt into directory named nested-directory. Can be used with files or directories.
Rename mv my-file.txt her-file.txt Renames file named my-file.txt to her-file.txt by moving my-file.txt to the current directory with a different name. Can be used with files or directories.
List as tree tree List all the files and directories and their nested files and directories in a tree like structure.
